Projekt wykonałem w architekturze MVC dla najlepszej przejrzystości kodu
Funkcjonalność:
1. Renderowanie produktów za pomocą skryptu (pobieranie danych z obiektu- symulacja uzyskania promisa z API tak jak by to było w prawdziwej aplikacji)
2. Działający koszyk z wyborem rozmiaru produktu i mozliwością wyczyszczenia z koszyka
3. Slider z timeoutem i kropkami nawigacji
4. Menu z podkategoraimi
5. Formularz kontaktowy
6. RWD
